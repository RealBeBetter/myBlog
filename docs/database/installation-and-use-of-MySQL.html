    【MySQL】MySQL的安装和使用

    雨下一整晚Real 9/12/2021 DatabaseMySQL

    # 一、安装准备

    安装之前，我们先登录 官网 (opens new window) 获取下载文件。

    选择相应系统版本之后，进入下一步的下载选项。

    其中，MySQL ZIP Archive是压缩版本，需要自己配置；MySQL MSI Installer是安装版本，安装过程中自动配置，我们下载这个版本使用。

    下面以 mysql-installer-web-community-8.0.22.0.msi 这个版本为例。

    # 二、安装步骤

    点击打开之后会有MySQL安装向导界面弹出。

    弹出这个界面之后，会紧接着弹出安装配置选项。这个时候需要进行选择。

    这个时候选择第一个Developer Default默认选项即可。点击下一步。

    勾选需要的组件之后，点击Execute进入下一步。

    进度条加载完毕之后，会弹出用户协议，勾选即可。

    之后成功之后进行勾选确定即可。这个时候会返回之前的界面，点击Next进入下一步。

    根据提示安装需要的组件。

    下载之后，会开始进行下一步的安装。

    等待安装完成之后，在安装完毕的页面中，点击Next进入下一页。

    在Product Configuration页面点击Next。

    之后在Type and Networking页面选择符合自己需要的即可，我们这里按照默认选择。

    在下面的页面中我们选择默认的选项即可，Next下一步。

    之后我们进入Accounts and Roles页面，开始创建root用户，设置root用户的密码。

    同时如果有添加User的需要，可以在右下角选择Add User的选项。

    设置密码之后，下面的页面按照默认选项即可。

    最后在Apply Configuration页面中确定，等待所有的步骤确认即可。

    到这一步为止，MySQL的安装就此完成，点击Finish即可完成。

    # 三、使用步骤

    点击Finish之后，MySQL会让用户确认之前的配置选项，完成之后进入下一步。

    这个时候，若是之前的步骤无误，直接输入之前设置的root用户的密码即可。点击check，会发现之前的选项后面会出现绿色的确认标志。点击下一步即可。

    之后进入下一个页面中，点击Execute等待完成点击Finish即可。

    安装完毕页面，会出现一个Copy Log to Clipboard的选项，选择是否复制日志到粘贴板，这个时候根据需要选择。

    1.1 DDL数据定义语言

    1)、定义:用于描述数据库中要存储实现实体的语言,简单说就是创建数据库和表、删除数据库和表、摧毁数据库和表

    2)、常见的关键字:CREATE、DROP、ALTER、TRUNCATE

    1.2 TPL事务处理语言

    1)、定义:它的语句能确保被DML语句影响的表的所有行及时得以更新

    2)、常见的关键字:BEGIN TRANSACTION，COMMIT、ROLLBACK

    1.3 DCL数据控制语言

    1)、定义:它的语句通过GRANT或REVOKE获得许可，确定单个用户和用户组对数据库对象的访问。某些RDBMS可用GRANT或REVOKE控制对表单个列的访问

    2)、常见的关键字:GRANT、REVOKE

    1.4 DML数据操作语言

    1)、定义:其语句包括动词INSERT，UPDATE和DELETE。它们分别用于添加，修改和删除表中的行

    2)、常见的关键字: INSERT、UPDATE、DELETE
